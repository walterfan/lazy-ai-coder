## i18n（vue-i18n）常见坑

- **避免在翻译字符串里直接放 JSON / 代码片段（包含 `{` `}`）**：`vue-i18n` 会把 `{...}` 当作占位符语法来编译，可能触发 message compilation error，导致页面渲染中断/空白。
- **需要展示 `{}` 时的处理方式**：
  - **优先方案**：不要走 i18n，把示例 JSON / Schema 放在组件里（常量 / 默认值），例如 textarea 的默认值（`v-model` 初始化）或普通字符串常量。
  - **必须走 i18n 时**：对 `{` `}` 做转义/拆分，避免被当作占位符解析（不要直接写成原始 `{ ... }`）。
    - 例：把 `{` `}` 分拆成字面量片段（例如 `{'{'} ... {'}'}` 这类写法），或改成不含花括号的展示形式。
